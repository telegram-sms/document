# Manual del Usuario

## Bienvenida

¡Gracias por usar Telegram SMS! Este manual te ayudará a configurar y usar el bot de Telegram SMS en tu dispositivo Android.

## Introducción

Telegram SMS es un bot de Telegram que se ejecuta en tu dispositivo Android. No requiere servidores intermediarios de terceros, pero tu red debe tener acceso a `api.telegram.org`. Puede administrar convenientemente tus SMS, llamadas perdidas e información de batería mediante comandos remotos.

## Enlaces rápidos

- [Generador de configuración](https://config.telegram-sms.com/) - Generar códigos QR para configuración fácil
- [Repositorio de GitHub](https://github.com/telegram-sms/telegram-sms) - Ver código fuente
