# Копирование (Carbon Copy)

Функция копирования пересылает копию событий, которые обрабатывает Telegram SMS, — входящие SMS, пропущенные звонки, уведомления приложений и оповещения о батарее — на другие сервисы push-уведомлений, в дополнение к отправке в Telegram. Популярные адресаты: Bark, Gotify, PushDeer, Lark и любой универсальный веб-хук.

Каждый адресат описывается запросом **HAR (HTTP Archive)**, который приложение воспроизводит при каждом событии. Поскольку запрос воспроизводится без изменений, поддерживается практически любой сервис push-уведомлений на основе HTTP без отдельной интеграции.

## Открыть настройки копирования

1. Откройте Telegram SMS и нажмите меню переполнения (⋮) → **Копирование**.
2. Вы увидите список настроенных сервисов копирования (при первом запуске он пуст).

## Добавить сервис

1. Нажмите плавающую кнопку **+**.
2. Вставьте HAR JSON сервиса в редактор. Редактор проверяет JSON по мере ввода и показывает ошибку, если структура недопустима.
3. Нажмите **OK**, чтобы сохранить.

Также можно нажать пункт **Сканировать** в меню переполнения, чтобы импортировать сервис с помощью QR-кода.

> Самый простой способ получить HAR с правильной структурой для распространённых провайдеров (Bark, Gotify, PushDeer, универсальный запрос cURL и т. д.) — это [Генератор конфигурации](https://config.telegram-sms.com/). Если вы хотите составить HAR вручную, см. [справочник по провайдерам копирования](https://dev.telegram-sms.com/CarbonCopyProvider) для разработчиков.

### Подстановочные поля

В теле HAR можно использовать следующие подстановочные поля. Они заменяются для каждого события перед отправкой запроса:

- `{{Title}}` — заголовок сообщения (например, тип события или отправитель SMS).
- `{{Message}}` — полный текст сообщения.
- `{{Copy}}` — вариант содержимого, удобный для копирования.

## Выбрать, какие события копируются

Набор источников событий — это **глобальная** настройка копирования, которая применяется ко всем сервисам. Откройте меню переполнения и выберите пункт настройки копирования, затем включите источники, которые хотите пересылать:

- Полученные SMS
- Пропущенные звонки
- Уведомления приложений
- Оповещения о батарее

На ваши сервисы копирования пересылаются только источники, включённые здесь.

## Изменить, включить, отключить или удалить сервис

- Нажмите сервис в списке, чтобы **изменить** его HAR или **удалить** его.
- В каждой записи указано её текущее состояние: **(Включено)** или **(Отключено)**.

## Отправить тест

Используйте пункт **Отправить тест** в меню переполнения, чтобы одновременно отправить тестовое сообщение на все настроенные сервисы и убедиться в доставке, прежде чем полагаться на неё. Если ещё не настроено ни одного сервиса, приложение сообщит, что проверять нечего.

## Импорт и шифрование

Пункт **получить конфигурацию** позволяет загрузить конфигурацию копирования с сервера, введя ID и пароль. Если конфигурация зашифрована, Telegram SMS расшифрует её с помощью указанного вами пароля (libsodium SecretBox), благодаря чему учётные данные, встроенные в ваш HAR, остаются защищёнными при передаче.
