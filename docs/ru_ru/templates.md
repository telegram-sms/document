# Шаблоны сообщений

Шаблоны позволяют настраивать формулировки и оформление сообщений, которые отправляет вам Telegram SMS. Каждый тип уведомления — входящее SMS, MMS, исходящее SMS, звонки, уведомления приложений и оповещения о батарее — имеет собственный шаблон, который можно свободно переписать.

## Открыть редактор шаблонов

Откройте Telegram SMS и нажмите меню переполнения (⋮) → **Шаблон**.

Вы увидите карточку для каждого типа сообщения. Каждая карточка показывает **предпросмотр в реальном времени** на основе примерных данных, поэтому вы сразу видите, как будут выглядеть ваши изменения.

## Изменить шаблон

1. Нажмите карточку того типа сообщения, который хотите изменить.
2. Отредактируйте текст в диалоговом окне. Используйте токены `{{Поле}}` (см. ниже) там, где нужно подставить реальное значение.
3. Нажмите одну из кнопок:
   - **OK** — сохранить свой шаблон.
   - **Сбросить** — отменить настройку и восстановить встроенное значение по умолчанию.
   - **Отмена** — закрыть без изменений.

Для переноса строки используйте `\n`, как и в шаблонах по умолчанию.

## Подстановочные поля

Подстановочное поле записывается как `{{Имя}}` и заменяется реальным значением при отправке сообщения. Любое удалённое поле просто не появится; текст за пределами полей сохраняется как есть. Каждый тип шаблона поддерживает свой набор полей:

| Шаблон | Доступные поля |
| --- | --- |
| Полученное SMS | `{{SIM}}`, `{{From}}`, `{{Content}}`, `{{Time}}` |
| Полученное MMS | `{{SIM}}`, `{{From}}`, `{{Subject}}`, `{{Content}}`, `{{Time}}` |
| Отправка SMS | `{{SIM}}`, `{{To}}`, `{{Content}}` |
| Входящий звонок | `{{SIM}}`, `{{From}}` |
| Пропущенный звонок | `{{SIM}}`, `{{From}}` |
| Уведомление | `{{APP}}`, `{{Title}}`, `{{Description}}` |
| Батарея | `{{BatteryLevel}}`, `{{Message}}` |

> `{{SIM}}` имеет смысл только на устройствах с двумя SIM-картами; на устройстве с одной SIM-картой оно заменяется пустым значением.

## Пример

Шаблон **Полученное SMS** по умолчанию:

```
[{{SIM}}Receive SMS]
From: {{From}}
Content: {{Content}}
```

Если вы хотите добавить время и собственный префикс, можно изменить его так:

```
📩 Новое SMS {{SIM}}
{{Time}}
{{From}} пишет:
{{Content}}
```

Нажмите **OK**, затем отправьте себе тестовое SMS, чтобы проверить новый формат.
