# Руководство пользователя

## Добро пожаловать

Благодарим вас за использование Telegram SMS! Это руководство поможет вам настроить и использовать бота Telegram SMS на устройстве Android.

## Введение

Telegram SMS - это бот Telegram, работающий на вашем устройстве Android. Он не требует сторонних промежуточных серверов, но ваша сеть должна иметь доступ к `api.telegram.org`. С его помощью вы можете удобно управлять своими SMS-сообщениями, пропущенными звонками и информацией об уровне заряда батареи с помощью удаленных команд.
