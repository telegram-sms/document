# 配置產生器

**配置產生器**（https://config.telegram-sms.com）是一個免費的網頁工具，可在電腦上建立 Telegram SMS 的配置，並以 QR Code 的形式傳到手機——這樣就不必在小螢幕上手動輸入像 Bot Token 這樣的長字串。

它完全在你的瀏覽器中執行，不需要註冊帳號。你的 Bot Token 不會離開本機裝置：它會在本機被編碼為 QR Code，產生器本身不會將它傳送到伺服器。

## 何時使用

- 想輸入很長的 **Bot Token**，又不想在手機上手動鍵入。
- 需要為多支手機設定相似的配置。
- 較習慣在有完整鍵盤的裝置上填寫表單。

> 如果你想把一支手機上*已經設定好*的完整配置複製到另一支手機，請使用 App 內的[傳輸配置](./transfer-config)。配置產生器則是用來在瀏覽器中從頭建立配置。

## 使用步驟

1. 在電腦（或另一支手機）上開啟 https://config.telegram-sms.com。
2. 填寫配置欄位——至少要填入你的 **Bot Token**。
3. 頁面會依填寫的內容產生一個 **QR Code**。
4. 在目標手機的 Telegram SMS 主配置畫面，點選**掃描**（相機）按鈕。
5. 掃描電腦螢幕上的 QR Code，對應的欄位會自動填好。
6. 完成其餘步驟（例如 `取得最近的對話 ID`），然後點選**測試並儲存**。

這與用來掃描[傳輸配置](./transfer-config)所產生 QR Code 的是同一個 App 內掃描器。

## 請妥善保管 QR Code

> ⚠️ 配置 QR Code 可能以明文形式包含你的 **Bot Token**。任何掃描它的人都能控制你的 Bot。請勿公開張貼或在截圖中分享，用完後請關閉頁面。
