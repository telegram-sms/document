# 消息模板

模板讓您可以自訂 Telegram SMS 發送給您的消息的措辭和排版。每一種通知類型——收到的短信、彩信、發出的短信、來電、應用通知和電池提醒——都有各自的模板，您可以自由改寫。

## 打開模板編輯器

打開 Telegram SMS，點擊溢出選單（⋮）→ **模板**。

您會看到每種消息類型對應的一張卡片。每張卡片都會用示例數據顯示**即時預覽**，讓您能直接看到修改後的效果。

## 編輯模板

1. 點擊您想修改的消息類型對應的卡片。
2. 在彈出的對話框中編輯文本。在您希望插入真實值的位置使用 `{{佔位符}}` 標記（見下文）。
3. 點擊下列按鈕之一：
   - **確定** — 儲存您的自訂模板。
   - **重置** — 放棄自訂並恢復內建預設值。
   - **取消** — 不做任何更改並關閉。

使用 `\n` 表示換行，與預設模板中的寫法相同。

## 佔位符

佔位符寫作 `{{名稱}}`，會在消息發送時被替換為真實值。您刪除的任何佔位符都不會出現；佔位符以外的文本會原樣保留。每種模板類型支持不同的佔位符集合：

| 模板 | 可用佔位符 |
| --- | --- |
| 收到的短信 | `{{SIM}}`、`{{From}}`、`{{Content}}`、`{{Time}}` |
| 收到的彩信 | `{{SIM}}`、`{{From}}`、`{{Subject}}`、`{{Content}}`、`{{Time}}` |
| 發送短信 | `{{SIM}}`、`{{To}}`、`{{Content}}` |
| 來電 | `{{SIM}}`、`{{From}}` |
| 未接來電 | `{{SIM}}`、`{{From}}` |
| 通知 | `{{APP}}`、`{{Title}}`、`{{Description}}` |
| 電池 | `{{BatteryLevel}}`、`{{Message}}` |

> `{{SIM}}` 僅在雙卡設備上有意義；在單卡設備上它會被解析為空值。

## 示例

預設的**收到的短信**模板為：

```
[{{SIM}}Receive SMS]
From: {{From}}
Content: {{Content}}
```

如果您想加入時間和自訂前綴，可以改成：

```
📩 新短信 {{SIM}}
{{Time}}
{{From}} 發來：
{{Content}}
```

點擊 **確定**，然後給自己發送一條測試短信以確認新格式。
