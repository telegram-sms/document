# 配置生成器

**配置生成器**（https://config.telegram-sms.com）是一个免费的网页工具，可在电脑上构建 Telegram SMS 的配置，并以二维码的形式传到手机——这样就不必在小屏幕上手动输入像 Bot Token 这样的长字符串。

它完全在你的浏览器中运行，无需注册账号。你的 Bot Token 不会离开本地设备：它会在本地被编码为二维码，生成器本身不会把它发送到服务器。

## 何时使用

- 想输入很长的 **Bot Token**，而不愿在手机上手动键入。
- 需要为多台手机配置相似的设置。
- 更习惯在带完整键盘的设备上填写表单。

> 如果你想把一台手机上*已经配置好*的完整设置复制到另一台手机，请使用 App 内的[传输配置](./transfer-config)。配置生成器用于在浏览器中从零开始构建配置。

## 使用步骤

1. 在电脑（或另一台手机）上打开 https://config.telegram-sms.com。
2. 填写配置项——至少要填入你的 **Bot Token**。
3. 页面会根据填写内容生成一个**二维码**。
4. 在目标手机的 Telegram SMS 主配置界面，点击**扫描**（相机）按钮。
5. 扫描电脑屏幕上的二维码，对应的字段会自动填好。
6. 完成其余步骤（例如 `获取最近的会话 ID`），然后点击**测试并保存**。

这与用来扫描[传输配置](./transfer-config)所生成二维码的是同一个 App 内扫描器。

## 请妥善保管二维码

> ⚠️ 配置二维码可能以明文形式包含你的 **Bot Token**。任何扫描它的人都能控制你的 Bot。请勿公开发布或在截图中分享，用完后请关闭页面。
