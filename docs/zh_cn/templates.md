# 消息模板

模板让您可以自定义 Telegram SMS 发送给您的消息的措辞和排版。每一种通知类型——收到的短信、彩信、发出的短信、来电、应用通知和电池提醒——都有各自的模板，您可以自由改写。

## 打开模板编辑器

打开 Telegram SMS，点击溢出菜单（⋮）→ **模板**。

您会看到每种消息类型对应的一张卡片。每张卡片都会用示例数据显示**实时预览**，让您能直接看到修改后的效果。

## 编辑模板

1. 点击您想修改的消息类型对应的卡片。
2. 在弹出的对话框中编辑文本。在您希望插入真实值的位置使用 `{{占位符}}` 标记（见下文）。
3. 点击下列按钮之一：
   - **确定** — 保存您的自定义模板。
   - **重置** — 放弃自定义并恢复内置默认值。
   - **取消** — 不做任何更改并关闭。

使用 `\n` 表示换行，与默认模板中的写法相同。

## 占位符

占位符写作 `{{名称}}`，会在消息发送时被替换为真实值。您删除的任何占位符都不会出现；占位符以外的文本会原样保留。每种模板类型支持不同的占位符集合：

| 模板 | 可用占位符 |
| --- | --- |
| 收到的短信 | `{{SIM}}`、`{{From}}`、`{{Content}}`、`{{Time}}` |
| 收到的彩信 | `{{SIM}}`、`{{From}}`、`{{Subject}}`、`{{Content}}`、`{{Time}}` |
| 发送短信 | `{{SIM}}`、`{{To}}`、`{{Content}}` |
| 来电 | `{{SIM}}`、`{{From}}` |
| 未接来电 | `{{SIM}}`、`{{From}}` |
| 通知 | `{{APP}}`、`{{Title}}`、`{{Description}}` |
| 电池 | `{{BatteryLevel}}`、`{{Message}}` |

> `{{SIM}}` 仅在双卡设备上有意义；在单卡设备上它会被解析为空值。

## 示例

默认的**收到的短信**模板为：

```
[{{SIM}}Receive SMS]
From: {{From}}
Content: {{Content}}
```

如果您想加入时间和自定义前缀，可以改成：

```
📩 新短信 {{SIM}}
{{Time}}
{{From}} 发来：
{{Content}}
```

点击 **确定**，然后给自己发送一条测试短信以确认新格式。
