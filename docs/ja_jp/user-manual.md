# ユーザーマニュアル

## ようこそ

Telegram SMSをご利用いただき、ありがとうございます！このマニュアルは、Androidデバイスで Telegram SMS ボットをセットアップおよび使用するのに役立ちます。

## はじめに

Telegram SMS は、Androidデバイス上で動作する Telegram ボットです。サードパーティのリレーサーバーを必要としませんが、ネットワークが `api.telegram.orz` にアクセスできる必要があります。リモートコマンドを使用して、SMS、不在着信、およびバッテリー情報を便利に管理できます。

## クイックリンク

- [設定ジェネレーター](https://config.telegram-sms.com/) - 簡単なセットアップ用の QR コードを生成

- [Telegram チャネル](https://t.me/tg_sms_changelog_eng) - 更新情報を購読
