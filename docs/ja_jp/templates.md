# メッセージテンプレート

テンプレートを使うと、Telegram SMS から届くメッセージの文言やレイアウトを自由にカスタマイズできます。各通知タイプ（受信 SMS、MMS、送信 SMS、着信、アプリ通知、バッテリー警告）には、それぞれ自由に書き換えられる専用のテンプレートがあります。

## テンプレートエディターを開く

Telegram SMS を開き、オーバーフローメニュー（⋮）→ **テンプレート** をタップします。

メッセージタイプごとにカードが表示されます。各カードはサンプルデータを使った**ライブプレビュー**を表示するため、変更後の見た目をその場で確認できます。

## テンプレートを編集する

1. 変更したいメッセージタイプのカードをタップします。
2. ダイアログ内のテキストを編集します。実際の値を挿入したい位置に `{{プレースホルダー}}` トークン（下記参照）を使います。
3. 次のいずれかをタップします：
   - **OK** — カスタムテンプレートを保存します。
   - **リセット** — カスタマイズを破棄し、組み込みの既定値に戻します。
   - **キャンセル** — 何も変更せずに閉じます。

改行には、既定のテンプレートと同じく `\n` を使用します。

## プレースホルダー

プレースホルダーは `{{名前}}` の形式で記述し、メッセージ送信時に実際の値に置換されます。削除したプレースホルダーは単に表示されません。プレースホルダー以外のテキストはそのまま保持されます。テンプレートの種類ごとに使用できるプレースホルダーが異なります：

| テンプレート | 使用できるプレースホルダー |
| --- | --- |
| 受信 SMS | `{{SIM}}`、`{{From}}`、`{{Content}}`、`{{Time}}` |
| 受信 MMS | `{{SIM}}`、`{{From}}`、`{{Subject}}`、`{{Content}}`、`{{Time}}` |
| 送信 SMS | `{{SIM}}`、`{{To}}`、`{{Content}}` |
| 着信 | `{{SIM}}`、`{{From}}` |
| 不在着信 | `{{SIM}}`、`{{From}}` |
| 通知 | `{{APP}}`、`{{Title}}`、`{{Description}}` |
| バッテリー | `{{BatteryLevel}}`、`{{Message}}` |

> `{{SIM}}` はデュアル SIM デバイスでのみ意味を持ちます。シングル SIM デバイスでは空の値に解決されます。

## 例

既定の**受信 SMS** テンプレートは次のとおりです：

```
[{{SIM}}Receive SMS]
From: {{From}}
Content: {{Content}}
```

時刻とカスタムの接頭辞を追加したい場合は、次のように変更できます：

```
📩 新着 SMS {{SIM}}
{{Time}}
{{From}} より：
{{Content}}
```

**OK** をタップしてから、自分宛てにテスト SMS を送って新しい書式を確認してください。
